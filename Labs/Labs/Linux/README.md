## 📖 Introduction

### What is Linux?
Linux is a free, open-source operating system used to run most of the world's servers, including AWS cloud infrastructure. It's the backbone of cloud computing — almost every AWS service runs on Linux behind the scenes.

### Why Do Cloud Engineers Need Linux?
- ☁️ Most AWS EC2 instances run Linux (Amazon Linux, Ubuntu, etc.)
- 💰 Linux is free — no licensing costs
- 🔒 More secure and stable than many alternatives
- ⚡ Lightweight and fast — perfect for servers
- 🛠️ Full control via the command line
