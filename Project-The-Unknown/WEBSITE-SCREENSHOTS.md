# Website Screenshots & Navigation

## Overview
This page documents the visual layout and navigation flow of The Unknown restaurant website.

## Site Navigation Map

Homepage (index.html)
├── Book a Table (booking.html)
├── Order Online (order.html)
└── Confirmation (confirmation.html)

All pages link back to each other through the navigation bar.

## Navigation Flow

### User Journey 1: Booking a Table
1. Customer lands on Homepage
2. Clicks "Book Your Table"
3. Fills in the Booking Form
4. Clicks "Confirm Booking"
5. Arrives at Confirmation Page
6. Receives confirmation email (Future Phase)

### User Journey 2: Ordering Online
1. Customer lands on Homepage
2. Clicks "Order Online"
3. Selects items from the Menu
4. Fills in delivery details
5. Clicks "Place Order"
6. Arrives at Confirmation Page
7. Receives confirmation email (Future Phase)
