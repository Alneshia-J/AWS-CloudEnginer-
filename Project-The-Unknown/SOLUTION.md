# Our Solution

## Overview
Migrate The Unknown restaurant to AWS by building a static website that enables online bookings and orders with automated confirmations and centralised data storage.

## What We Built
A responsive static website hosted on Amazon S3 that allows customers to:
- Book a table online (date, time, guest count, seating preference)
- Place orders online (starters, mains, desserts, drinks)
- Receive automated confirmation of their booking or order

## How It Solves Each Problem

### Problem 1: Double Bookings & Order Mix-Ups
Solution: Digital forms capture bookings and orders accurately — no more miscommunication.

### Problem 2: No Online System
Solution: A fully functional website accessible 24/7 from any device.

### Problem 3: No Automated Confirmations
Solution: AWS Lambda sends automatic confirmation emails to customers after every booking or order.

### Problem 4: No Centralised Data
Solution: Amazon DynamoDB stores all customer data, bookings, and orders in one secure location.

### Problem 5: Costly Staff Expansion
Solution: Automation replaces the need for additional staff, saving R7,830–R14,830 per month.

## Key Benefits
- Scalability — handles traffic spikes without manual intervention
- Cost-effective — R15–R170/month vs R8,000–R15,000 for staff
- Enterprise-grade security — built on AWS infrastructure
- 99.99% uptime — always available for customers
- Data insights — track bookings, orders, and customer behaviour
- 24/7 accessibility — customers can book or order anytime
- Competitive advantage — modern digital presence
