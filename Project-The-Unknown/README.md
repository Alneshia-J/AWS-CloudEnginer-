# The Unknown — Restaurant Website (AWS Migration)

The Unknown is a popular restaurant in Johannesburg serving 200+ customers daily. This project builds a static website hosted on AWS to solve operational challenges like double bookings, order mix-ups, and lack of automated confirmations.

Built as a group project for the Praesignis AWS re/Start Programme | July 2025

## The Problem
- Double bookings and order mix-ups
- No online booking or ordering system
- No automated confirmations for customers
- No centralised data storage
- Costly staff expansion needed

## Our Solution
Migrate to AWS by building a static website that enables online bookings and orders with automated confirmations and centralised data storage

## Website Pages
- Homepage — Welcome, hero section, navigation
- Booking Page — Date, time, guest count, seating preference
- Order Page — Menu with starters, mains, desserts, drinks
- Confirmation Page — Success message and next steps

## Cost Savings
- AWS Solution: R15–R170/month
- Hiring Staff: R8,000–R15,000/month
- Monthly Savings: R7,830–R14,830
