# The Unknown - Restaurant Website

A restaurant website project for the Praesignis AWS re/Start programme.

## Project Description
The Unknown is a modern restaurant website designed and developed as a collaborative group project for the Praesignis AWS re/Start Programme. This project showcases our team's ability to build a responsive, user-friendly web application while applying cloud deployment skills using AWS services. The website features an interactive menu, reservation system, and a visually appealing design that brings "The Unknown" dining experience to life.
